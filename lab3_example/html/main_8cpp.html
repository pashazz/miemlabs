формирование вектора множеств dst из строки src по правилу

dst[i] это множество слов из строки src, начинающихся на одну букву; множества отсортированы в обратном алфавитном пор¤дке букв, с которых начинаются их слова.

Слова отсортированы в обратном алфавитном порядке букв.

Аргументы
[in]src- строка, которую нужно обработать
[out]dst- результирующий вектор (предполагается пустой на входе)

Пример

string src=”this is the malt that lay in the house that jack built”;
VectorOfStrings dst;
parseString(src,dst);
printVector(dst); // dst = {{“this”,”the”,”that”}, {“malt”}, {“lay”}, {“jack”}, {“is”,”in”}, {“house”}, {“built”}}
